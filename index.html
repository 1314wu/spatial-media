Overview

A collection of specifications and tools for 360° video and spatial audio.

Specifications

  • Spatial Audio metadata specification
  • Spherical Video metadata specification
  • Spherical Video V2 metadata specification

Tools

  • Spatial Media Metadata Injector for injecting spatial media metadata in media files.
  • Jump Inspector, an Android app for previewing VR videos with spatial audio.
